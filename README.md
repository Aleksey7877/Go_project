/webinars/hw2/ledger - задание 4, коммит 4.
Ledger дополнен интерфейсом валидации данных в транзакциях и бюджетах через метод Validate().
Программа разделена на:
-demo.go - хендлеры транзакций и бюджетов;
-io.go - чтение бюджетов из json-файла;
-main.go - запуск и проверка сценариев работы;
-models.go - структуры данных ledger;
-storage.go - бизнес-логика работы и записи в память транзакций и бюджетов;
-validation.go - интерфейс валидации значений транзакций и бюджетов.
